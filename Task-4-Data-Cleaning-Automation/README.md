Automated Employee Data Cleaning & Reporting System

Objective
Automate employee data cleaning and reporting workflows using Python.

Features
- Handling missing values
- Removing duplicates
- Data preprocessing
- Automated report generation
- Visualization of salary distribution
- Correlation heatmap generation
- Exporting cleaned datasets automatically

Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

Generated Outputs
- Cleaned employee dataset
- Summary report
- Missing value analysis
- Salary distribution visualization
- Correlation heatmap

Outcome
Learned automated data preprocessing, reporting workflows, and visualization techniques for efficient data analysis.

Generated Reports
The project automatically generates:
- Missing value analysis chart
- Salary distribution graph
- Correlation heatmap
- Statistical summary report
- Cleaned dataset export
