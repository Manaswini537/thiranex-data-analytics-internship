Customer Segmentation Project

Objective
Segment customers based on purchasing behavior and demographics using clustering techniques.

Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Power BI

Features
- Performed K-Means clustering
- Analyzed customer spending patterns
- Visualized customer segments
- Created an interactive Power BI dashboard

Outcome
Successfully identified different customer groups based on annual income and spending score.

Files Included
- Dataset
- Python clustering code
- Power BI dashboard
- Dashboard screenshots
